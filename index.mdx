## Architecture Highlights

Built with modern backend engineering principles:

- **PostgreSQL JSONB** for schema-flexible product metadata
- **LATERAL joins** and recursive CTEs for optimized queries
- **Schema-driven validation** with dynamic normalization pipeline
- **Soft deletion** with EF Core global query filters
- **Automatic SKU generation** based on category hierarchy
- **FluentValidation** for request validation
- **Compile-time DI** registration via source generators

<Note>
This project is actively under development. Authentication and inventory workflows are planned for future releases.
</Note>
