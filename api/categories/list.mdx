---
title: "List Categories"
api: "GET /api/v1/admin/categories"
description: "Retrieve all categories in the grocery store system"
---

This endpoint returns a complete list of all product categories, including their hierarchical relationships and nesting levels.

## Request

No parameters are required for this endpoint.

## Response

Returns an array of category objects with the following structure:

<ResponseField name="id" type="integer" required>
  Unique identifier for the category
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the category
</ResponseField>

<ResponseField name="parentId" type="integer | null">
  ID of the parent category. Null if this is a root-level category
</ResponseField>

<ResponseField name="fullPath" type="string | null">
  Complete hierarchical path to this category (e.g., "Food/Dairy/Cheese")
</ResponseField>

<ResponseField name="level" type="integer" required>
  Nesting level in the category hierarchy. Root categories have level 0
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.example.com/api/v1/admin/categories" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Example Response

```json
[
  {
    "id": 1,
    "name": "Food",
    "parentId": null,
    "fullPath": "Food",
    "level": 0
  },
  {
    "id": 2,
    "name": "Dairy",
    "parentId": 1,
    "fullPath": "Food/Dairy",
    "level": 1
  },
  {
    "id": 3,
    "name": "Cheese",
    "parentId": 2,
    "fullPath": "Food/Dairy/Cheese",
    "level": 2
  }
]
```

## Status Codes

<ResponseField name="200" type="OK">
  Successfully retrieved the list of categories
</ResponseField>

<ResponseField name="403" type="Forbidden">
  User does not have permission to access admin category endpoints
</ResponseField>
