---
title: 'List Products'
api: 'GET /api/v1/admin/products'
---

Retrieves all active products with their associated attributes and metadata.

## Response

Returns an array of product objects with their complete details.

<ResponseField name="Id" type="integer">
  The unique identifier of the product.
</ResponseField>

<ResponseField name="Name" type="string">
  The name of the product.
</ResponseField>

<ResponseField name="Price" type="decimal">
  The price of the product.
</ResponseField>

<ResponseField name="CategoryId" type="integer">
  The ID of the product's category.
</ResponseField>

<ResponseField name="BrandId" type="integer">
  The ID of the product's brand.
</ResponseField>

<ResponseField name="CountryId" type="integer">
  The ID of the product's country of origin.
</ResponseField>

<ResponseField name="Sku" type="string">
  The SKU (Stock Keeping Unit) identifier.
</ResponseField>

<ResponseField name="Description" type="string">
  The product description.
</ResponseField>

<ResponseField name="BaseUnit" type="string">
  The base unit for the product (e.g., "kg", "L", "pcs").
</ResponseField>

<ResponseField name="Metadata" type="array">
  An array of product attributes and their values.
  
  <Expandable title="Metadata Object">
    <ResponseField name="Name" type="string">
      The name of the attribute.
    </ResponseField>
    
    <ResponseField name="Value" type="string">
      The value of the attribute.
    </ResponseField>
    
    <ResponseField name="Unit" type="string">
      The unit of measurement for the attribute, if applicable.
    </ResponseField>
  </Expandable>
</ResponseField>

## Request Example

```bash
curl -X GET "https://api.example.com/api/v1/admin/products" \
  -H "Content-Type: application/json"
```

## Status Codes

- `200 OK` - Successfully retrieved product list
- `403 Forbidden` - Insufficient permissions to view products

## Notes

- This endpoint only returns **active** products (where `IsActive = true`)
- Soft-deleted products are automatically excluded from the results
- The metadata array contains denormalized attribute information with names and units for easy consumption
- Products are returned with all their attributes expanded for comprehensive product information
